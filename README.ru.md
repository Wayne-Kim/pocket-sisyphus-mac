# Pocket Sisyphus для Mac

[한국어](README.md) · [English](README.en.md) · [日本語](README.ja.md) · [العربية](README.ar.md) · [Español](README.es.md) · [Français](README.fr.md) · [हिन्दी](README.hi.md) · [Português (BR)](README.pt-BR.md) · **Русский** · [简体中文](README.zh-Hans.md)

> Приложение для macOS, позволяющее безопасно управлять с телефона несколькими CLI кодинг-агентов (Claude Code, Google Antigravity и др.), работающими на вашем Mac.
> На основе **onion-сервисов Tor** · **без внешних серверов**.

## 📥 Загрузка

Последний билд доступен на [странице Releases](../../releases/latest):

```
PocketSisyphusMac-v<версия>-build<номер>.dmg
```

DMG подписан Apple Developer ID и нотаризован Apple — Gatekeeper открывает его без предупреждений.

## 🧰 Системные требования

| Параметр | Минимум |
|---|---|
| macOS | 13.0 (Ventura) или новее |
| Процессор | Apple Silicon или Intel |
| Диск | ~400 МБ (включая встроенные Node.js + Tor) |
| Сеть | Любое интернет-соединение (без настройки firewall/роутера) |

## ⚡️ Установка одной строкой (рекомендуется)

Вставьте это в Терминал — последняя версия автоматически установится в `/Applications` и запустится. Никаких зависимостей (нужен только встроенный в macOS `curl`):

```bash
curl -fsSL https://raw.githubusercontent.com/Wayne-Kim/pocket-sisyphus-mac/main/install.sh | bash
```

Скрипт читает прямую ссылку на DMG из `appcast.xml` последнего релиза, затем скачивает → монтирует → копирует `.app` в `/Applications` → запускает. DMG нотаризован (notarize + staple), поэтому Gatekeeper не предупреждает. Опасаетесь `curl | bash`? Сначала прочитайте [install.sh](install.sh).

## 🚀 Ручная установка

1. Дважды кликните по `PocketSisyphusMac-v*.dmg`.
2. Перетащите **Pocket Sisyphus.app** в папку **Программы**.
3. Запустите **Pocket Sisyphus** через Launchpad или Spotlight.

> ⚠️ Если Gatekeeper выдаёт "неустановленный разработчик", значит билет нотаризации ещё не закеширован локально. Кликните правой кнопкой по иконке → **Открыть** один раз, дальше macOS будет доверять автоматически.

## 📱 Первый запуск — Сопряжение телефона

1. (Mac) При первом запуске daemon автоматически создаёт onion-сервис Tor (`xxxxx.onion`).
2. (Mac) Иконка в строке меню → **Show Pairing QR**.
3. (Телефон) При первом запуске iOS-приложение **Pocket Sisyphus** попросит выбрать **режим подключения** — «Откуда угодно (Tor)» или «Только та же Wi-Fi».
4. (Телефон) Отсканируйте QR для сопряжения.
   - **Откуда угодно** — телефон подключается к Mac через сеть Tor из любой точки (LTE/5G/Wi-Fi).
   - **Только та же Wi-Fi** — подключается напрямую по частному адресу только когда телефон и Mac находятся в одной Wi-Fi сети (быстрее), и блокирует соединение из внешних сетей.

После сопряжения вы можете общаться, подтверждать изменения (Approval) и просматривать diff'ы файлов с телефона.

## 🛡️ Модель безопасности

- **Никаких внешних серверов** — инфраструктуры мейнтейнера нет, только сеть Tor.
- **Не зависит от NAT/CGNAT** — обе стороны подключаются исходящим (outbound) трафиком к Tor, port-forwarding не нужен.
- **Двойная аутентификация** — адрес `.onion` (ключ Ed25519) + Bearer-токен.
- **Daemon слушает только 127.0.0.1** — напрямую недоступен, единственная точка входа — Tor onion.
- **Режим только той же Wi-Fi (опционально)** — когда включён, подключается только по частному адресу в той же Wi-Fi сети и блокирует Tor, публичные IP и весь внешний исходящий трафик (fail-closed). Для случаев, когда нужна гарантия, что ни один пакет не покинет вашу локальную сеть.

## 🧯 Устранение неполадок

**Приложение не открывается / предупреждение Gatekeeper**
→ Правый клик по иконке в `Программах` → Открыть → подтвердить.

**Нет иконки в строке меню**
→ Mission Control может держать её на другом рабочем столе. `cmd+space` → поиск "Pocket Sisyphus".

**Не строится Tor-цепочка**
→ Некоторые корпоративные/университетские сети блокируют Tor. Попробуйте через личную точку доступа.

**Сопряжение не работает**
→ В режиме «Откуда угодно (Tor)» телефон и Mac не обязаны быть в одной Wi-Fi сети (LTE/5G подойдёт), если iOS-приложение **Pocket Sisyphus.app** уже завершило загрузку Tor (прогресс виден внутри приложения). В режиме «Только та же Wi-Fi» телефон и Mac ОБЯЗАТЕЛЬНО должны быть в одной Wi-Fi сети.
